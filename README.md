 - Pure (S)CSS
 - Custom Animations

## ⚒ Installation/Setup
The installation requires at least **node version 10.4**

```Shell
# run in the root directory of the project
npm install
npm run dev
# run in the root directory of the project
yarn install
yarn run dev
```
