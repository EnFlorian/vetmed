# 🐶🐱🐢 VetMed

##### A 1-2 Page online presence for a veterinarian office.



## 📖 Features
 - Fully Responsive
 - Pure (S)CSS
 - Custom Animations
 - Clientside Server Submission

## ⚒ Installation/Setup
The installation requires at least **node version 10.4**

```Shell
# run in the root directory of the project
npm install
npm run start
# run in the root directory of the project
yarn install
yarn run start
```
